patches:
Description: All the patch information from the JSON object in the database

currentDevice:
Description: The current device that is being interacted with or managed within the application. Its initial value is null.

patchNumber:
Description: ID of the current patch

selectedFile:
Description: A file that a user has selected through file input for uploading.

selectedImage:
Description: An image file that the user has selected for uploading.

patchInfo:
Description: Initial information related to a series of patches. It is initialized with an empty array.

singlePatchInfo:
patchInfo for a single patch.
